# Freek Top 100

This project tracks Acorn formalizations of Freek Wiedijk's Top 100 theorem
list.

Source: https://www.cs.ru.nl/~freek/100/

The checked Acorn modules live under `src/top100/`, with certificates under
`build/top100/`. Only complete, verified theorem formalizations should be added
there. Staged lemmas, smoke tests, harness diagnostics, and work-in-progress
experiments should stay outside the checked Top 100 modules until they prove the
public theorem target.

Submissions generated by the automation workflow are marked `AixBot-generated`
in their pull requests.
