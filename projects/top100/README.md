# Freek Top 100

This directory collects Acorn formalizations of Freek Wiedijk's Top 100
theorem list.

Source: https://www.cs.ru.nl/~freek/100/

Only complete, verified theorem formalizations belong here. Staged lemmas,
smoke tests, harness diagnostics, and work-in-progress experiments should stay
outside this directory until they prove the public theorem target.

Submissions generated by the automation workflow are marked `AixBot-generated`
in their pull requests.
