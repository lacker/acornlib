# Probability Theory

Goal: provide the probability layer that sits on top of measure theory in Mathlib.

- [ ] Define probability measures and probability spaces
- [ ] Add random variables, pushforwards, and laws
- [ ] Develop expectation, variance, and moments
- [ ] Support independence of events and random variables
- [ ] Add conditional probability and conditional expectation
- [ ] Implement standard discrete distributions
- [ ] Implement standard continuous distributions
- [ ] Formalize convergence in probability, almost surely, in distribution, and in `L^p`
- [ ] Add the law of large numbers and central limit theorem
- [ ] Support martingales, stopping times, and basic stochastic-process results
