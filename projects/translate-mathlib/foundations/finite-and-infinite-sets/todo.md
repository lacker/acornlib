# Finite and Infinite Set Infrastructure

Goal: make finiteness a reusable notion across sets, algebra, and combinatorics.

Summary: list-backed finite sets, finite images, list-filter/list-set extraction lemmas, finite subset extraction from list-backed and existential finite witnesses, exact unique-list extraction from set and finite-set cardinality, basic cardinality-at-most transfer, exact-cardinality transfer for injective images, disjoint finite-set unions, remove/intersection/difference cancellation cases, intersection/difference partition additivity and subtraction forms, subgroup-to-set finiteness and cardinality transfer, finite-subgroup set views, pigeonhole-style collision lemmas for finite sets and lists, the infinite-set predicate API, and injective preimage lemmas for list-finite codomains are now available.
