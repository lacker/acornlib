# Core Type Constructions

Goal: provide stable APIs for the small family of basic type constructors that appear everywhere.

- [ ] Strengthen the API for pairs and product types
- [ ] Add sum type infrastructure and case-splitting lemmas
- [ ] Add option type infrastructure and useful elimination lemmas
- [ ] Support dependent pair constructions where Acorn syntax allows them
- [ ] Add map functions for product, sum, and option
- [ ] Add extensionality lemmas for pair-like structures
- [ ] Add associativity and commutativity equivalences for products and sums
- [ ] Add convenience lemmas for currying and uncurrying
- [ ] Add set- and function-level APIs that interact well with these constructors
- [ ] Refactor current pair-only code toward a general construction toolkit
