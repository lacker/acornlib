# Algebraic Geometry

- [ ] Define prime spectra and the Zariski topology

- [ ] Add projective spectrum and projective-space constructions
- [ ] Develop coherent sheaves, divisors, and line bundles
- [ ] Add dimension, regularity, and smoothness preliminaries
- [ ] Prove affine Nullstellensatz-style results
- [ ] Support cohomological infrastructure where needed by later geometry
