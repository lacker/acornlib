# Extensionality Lemmas for Sets, Predicates, and Functions

Goal: make equality proofs for basic set-like and function-like objects short and routine.

- [ ] Add a theorem `Set.ext` stating that two sets are equal when they have the same elements
- [ ] Add a theorem for predicate extensionality on `T -> Bool`
- [ ] Add a theorem for unary function extensionality from pointwise equality
- [ ] Add a theorem for binary function extensionality from pointwise equality
- [ ] Add subset-antisymmetry lemmas that reduce set equality to two inclusions
- [ ] Add extensionality lemmas for finite sets via their underlying sets
- [ ] Add extensionality lemmas for subgroups via their membership predicates
- [ ] Add extensionality lemmas for structures whose only data is a function field
- [ ] Add a small collection of rewrite lemmas that turn `x ∈ s` equalities into function equalities
- [ ] Audit existing proofs and replace ad hoc extensionality arguments with the new theorems
